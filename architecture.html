04 · /generate-report Pipeline — 11 Steps  (app/api/report.py)
1
Cache Check
Query health_scores <24h old. Return cached unless force=True
2
Biz Lookup
SELECT * FROM businesses WHERE id=? → 404 if not found
3
Google Places
fetch_all_data(place_id, category) → rating, 5 reviews, competitors. Skip if manual_ prefix
4
Apify Augment
get_reviews(max=50) — bypasses Google 5-review cap. Non-fatal on failure
5
Competitor Pipeline
competitor_pipeline.run() — Nearby+distance search → Haiku tag → brand top-up → Apify (parallel) → Cohere centroid → cosine ≥ 0.55. 7d cache + manual pins
6
POS Signals
pos_signals(biz_id, days=180) → revenue_trend, slow_cats, aov, repeat_rate. Never raises
7
Classify Reviews
classify_reviews() via Haiku → sentiment 1–5 + dominant_complaint topic. Non-fatal
8
Sub-scores
review_score() + competitor_score() + pos_score() → final_score + band (healthy/watch/at_risk)
9
Insights
generate_insights() via Claude Sonnet → 3–6 insights + 1 action. Retry ×1 on JSON fail
10
Competitor Analysis
analyze_competitors() via Sonnet — themes (what competitors are praised for) + opportunities. Cached Apify reviews. Non-fatal
11
Persist
INSERT INTO health_scores (scores, insights, action, report_payload). Non-fatal on fail

06 · Health Score Engine  —  app/services/health_score.py
final_score  =  review_score × 0.40  +  competitor_score × 0.25  +  pos_score × 0.35
Reviews  ·  40%
Competitors  ·  25%
POS Sales  ·  35%
🟢 Healthy  ≥ 80
🟡 Watch  60 – 79
🔴 At Risk  < 60
⭐
review_score( ) → 0–100
Quality · Velocity · Sentiment trend
Quality Points0 – 55 pts
((rating − 1) / 4.0) × 55  ·  Normalised within actual [1–5] Google scale. 1★ → ~0 pts.
Velocity / Volume Points0 – 25 pts
If Apify data: reviews/month over 6m window via compute_velocity(). Full marks at 8/month.
Fallback (raw Google): log₁₀(total_reviews) × 10
Sentiment Trend Points0 – 20 pts
If classified: Claude Haiku sentiment_avg / 5 × 20 (catches polite 4★ with negative text).
Fallback: star_avg / 5 × 20 over last 50 reviews.
🏁
competitor_score( ) → 0–100
vs. similarity-filtered competitors (Cohere)
60 + (my_rating − mean_competitor) × 30
Clamped to[0, 100]
No competitors found65 (neutral)
Sourcecompetitor_pipeline.run()
Similarity threshold≥ 0.55 cosine
Max competitors10 · 7-day cache
Manual pins via /competitors/{id} always lead the list, never expire
💹
pos_score( ) → 0–100
4 POS signals · category-aware thresholds
Revenue Trend0 – 40 pts
Category-specific bands. Restaurant: floor −30% → full +15%. Retail: floor −40% → full +20%. No data → neutral 50.
Inventory Health0 – 25 pts
Slow categories: 0 slow → 25, 1 slow → 17, 2 slow → 8, 3+ → 0. Min ₹50/day to qualify.
AOV Direction0 – 15 pts
rising → 15, stable → 9, falling → 4. Threshold: ±5%.
Repeat Customer Rate Trend0 – 20 pts
trend >+5% → 20 pts. −5%–+5% → 14. −15%–−5% → 7. <−15% → 2. No data → 10 (neutral).
